About The Project

UNITY SNOBAL PROJECT

A Virtual Reality Tutorial For HTC Vive Controllers

The main purpose of this project is to create a tutorial that will articulate the full potential of virtual reality, provide guidelines around the use of the HTC Vive controllers, and demonstrate the emotional intensity of VR

The tutorial is developed using the unity game engine and C# for scripting. The tutorial consisted of a scene where a robotic AI teaches the user how to use the controllers by instructing the them through audio and text of which button to press and explaining the fuctionalities of each button. The AI will also provide the user different items, each with different interaction characteristic to demostrate what the user can do within the VR world. Verticality of the scene and the way the tutorial ends is also expected to bring some emotions out form the user.

As a programmer my main task in this project, along with three other programmer, is to develop scripts that will turn the designs and concepts, and models into a reality within the vertual space. These includes but are not limited to, making game objects interactabel, sound scripting, creating certain conditions that will trigger certain events, making the AI speak to the user accompanied by captions to esure users fully understand, etc.

Snobal Project Contributions (deliverables)

GameControl.cs

The GameControl script is the main script that controls the sequence of the tutorial spanning from different stages of the tutorial to include audio and text instructions being conveyed by the AI to the user, to accessing different scripts throughout the files that will trigger certain stages.

This script is a collaboration among the programmers with comments of which blocks each programmer is responsible for. My main work in this script was to control the AI's voice and caption throughout various stages.

Download files

AIbehaviour.cs

The AI behaviour script controls the AI's movement to include constantly facing the the user, moivng to where the user is looking at the beginning the the tutorial and returning to its original position to commence the tutorial.

Download Files

TransparentBallScript.cs

This scripts disables the physics attributes of objects when inside the transparent sphere thus making them float. Once taken outside the sphere, objects retain their physics attributes. Each objects, once taken out of the sphere also instantiate related game objects like basketball ring for the basket ball and target for bow and arrow.

TransparentBallMovement.cs

This script contains a function that moves the transparent sphere along with objects inside of it. The function is called by the GameControl script when a certain contidion is met during the tutorial. This effect was intended to give the user an unusual experience.
